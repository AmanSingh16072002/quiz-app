# Quiz Management System (Full Stack)

## 🚀 Overview
A full-stack quiz platform with CMS (admin) and student interface.

## 🛠 Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MongoDB Atlas

## ✨ Features
- User authentication (Admin & Student)
- Create and manage quizzes
- Attempt quizzes with real-time saving
- Automatic score calculation
- Detailed report with correct/incorrect answers

## 📂 Project Structure
- client/ → frontend
- server/ → backend

## ⚙️ Setup Instructions

### Backend
cd server  
npm install  
npm run dev  

### Frontend
cd client  
npm install  
npm run dev
